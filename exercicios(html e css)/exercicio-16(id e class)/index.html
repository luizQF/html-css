Diferença entre ID e class

Identificadores ID

O id é um identificador único, ou seja só pode ter um com o mesmo nome, as alterações indicadas com o nome do ID só irão afetar o ID

Identificadores Class

A class pode ser utilizava em multiplas tags, e podem ter o mesmo nome, as alterações indicadas irão afetar todos os elementos com o nome da class

<Div class="">

A div funciona como uma caixa com tudo que vai ser alterado, basta criar uma div com os elementos dentro e definir uma classe a ela, todos as alterações feita na div afetara todos os itens.

elementos com mesma tag podem ter um id dentro da div, para atribuir estilos diferentes dos demais
