Tags e suas respectivas funções

tag ins pode ser usada para sublinhar uma palavra ex:Palavras sublinhada

tag strong usada para deixar a palavra em negrito exemplo palavra em negrito

tag code pode ser usada para mudar a forma da palavra ex : palavra em formato de code
se for somada com tag pre o texto será reproduzido exatamente como esta no script

tag em utilizada para deixar a palavra em italico ex: palavra em italico

tag del pode ser usada para desconsiderar um texto exemplo não deve ser considerado

a tag "mark" pode ser utilizada para marcar um texto como se fosse um marca texto, exemplo: texto marcado

bdo dir:rtl"direita pra esquerda"/ltr"esquerda pra direita" serve para inverter o texto exemplo: texto invertido
